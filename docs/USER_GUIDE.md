# AETHER-1 User Guide

## Welcome to the Future

AETHER-1 is a **living digital organism** — not just another AI tool.

### Quick Start (Windows - One Click)
1. Download `AETHER1_Setup.exe` from Releases
2. Run the installer (one click)
3. Desktop shortcut appears
4. Double-click to launch the Hacker GUI
5. The system auto-trains a frontier SOTA model on first run

### The Hacker GUI
- **Live Status Panel**: Real-time ticks, free energy, active goals
- **Memory Matrix**: Hierarchical memory visualization
- **World Model**: Active inference predictions
- **Action Console**: Live actions taken
- **Research Suite**: Explore arXiv trends, run experiments
- **Training Suite**: Train / fine-tune / evolve the model
- **Testing Suite**: Run benchmarks and safety tests

All interfaces are self-explanatory with tooltips and hacker-style terminal aesthetics.
